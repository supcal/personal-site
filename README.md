# 李超个人学术主页

这是一个托管在 GitHub Pages 上的静态个人学术主页。

## 内容来源

页面基础信息整理自天津师范大学计算机与信息工程学院教师主页：

https://xxgc.tjnu.edu.cn/info/1166/3643.htm

## 文件

- `index.html`：主页内容
- `data.json`：可视化编辑和主页渲染使用的数据
- `admin.html`：可视化编辑页面
- `styles.css`：页面样式
- `script.js`：年份更新脚本

## 可视化编辑

打开 `admin.html` 后可以通过表单编辑主页内容。编辑完成后下载或复制新的 `data.json`，再到 GitHub 仓库替换同名文件，GitHub Pages 会自动重新发布。
